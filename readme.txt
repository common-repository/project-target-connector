This is a plugin which connects your store with your Project Target application.

== Description ==

This plugin is relying on a 3rd party service Project Target and it can't work without it. 
Project Target is a software solution that enables you to monitor and control sales performance and also makes you stay focused all the time in order to achieve daily, monthly, or yearly targets.

The plugin will send the following data to the Project Target application:
Customer First and Last name
Customer Email address
Product name and quantity
Product Price.
